# AI-Powered Coffee Recommendation Mobile Application

This project aims to develop an intelligent mobile application that revolutionizes the coffee ordering experience by solving a critical pain point: customer indecision and lack of knowledge about coffee parameters. Through an AI-powered conversational interface, the system helps users discover personalized coffee recommendations based on their preferences, taste profiles, and behavior patterns.

## Features
- AI-powered coffee recommendations
- Conversational user interface
- Personalized suggestions based on user preferences
- Microservices-based backend architecture

## Project Goal
To enhance customer experience in coffee ordering by reducing decision fatigue and improving personalization using artificial intelligence.
